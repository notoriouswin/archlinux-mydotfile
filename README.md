## 📦 Instalação

Clone o repositório:

```bash
git clone git@github.com:notoriouswin/archlinux-mydotfile.git
cd archlinux-mydotfile
```

Copie os arquivos para o diretório home:

```bash
cp -r . ~/
```

Ou crie links simbólicos:

```bash
ln -sf ~/archlinux-mydotfile/.zshrc ~/.zshrc
```
